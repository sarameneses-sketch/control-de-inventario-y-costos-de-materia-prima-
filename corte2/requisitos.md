Requerimientos Funcionales  
El sistema permitirá el registro y login de usuarios
El sistema mostrará un catálogo detallado de los servicios de marketing y publicidad que ofrece la empresa
El sistema debe procesar pagos en línea de forma segura y rapida
El sistema permitirá a los administradores gestionar el estado de los proyectos contratados y poder revisarlos

Requerimientos No Funcionales  
Rendimiento: El sistema debe cargar en menos de 3 segundos
Disponibilidad: al app debe estar disponible y operando el 99% del tiempo 
Usabilidad: La interfaz debe ser totalmente intuitiva es decir que permita la rapidez a la hora del usuario realizar el pedido
Seguridad: Los datos de los clientes deben estar protegidos bajo normativas de privacidad de datos
Escalabilidad: El sistema debe permitir ingresar al mismo tiempo al usuario y al administrador sin colapsar
