## Introducción
Este documento detalla los requerimientos necesarios para el desarrollo de la plataforma de servicios de marketing.

## Requerimientos Funcionales (RF)
| ID | Descripción |
| :--- | :--- |
| RF-01 | El sistema permitirá el registro y login de usuarios. |
| RF-02 | El sistema mostrará un catálogo detallado de servicios de marketing. |
| RF-03 | El sistema debe procesar pagos en línea de forma segura. |
| RF-04 | El sistema permitirá a los administradores gestionar el estado de los proyectos contratados. |

## Requerimientos No Funcionales (RNF)
| ID | Descripción |
| :--- | :--- |
| RNF-01 | La aplicación debe cargar en menos de 3 segundos. |
| RNF-02 | La interfaz debe ser totalmente responsiva (móvil y escritorio). |
| RNF-03 | Los datos de los clientes deben estar protegidos bajo normativas de privacidad de datos. |
